# Troubleshooting

## I cannot see a recurring task

Open the **recurring tab** and check that the recurring task is enabled and not archived. Daily, weekly, and monthly recurring tasks appear in their matching planning tab and view. Someday presets do not create tasks automatically; use the add action on the preset.

Also check the relevant tab and view, including the Tomorrow view or a previous time-period view if the task was moved or scheduled there.

## I moved a task by mistake

If the undo message is still visible, tap **Undo**. Otherwise, open the Task Info sheet or use the available swipe action to move the task back to the intended timeframe.

## I deleted or archived a task

Open the Settings sheet, then select **Trash**.

- Restore an archived task from the Archived list.
- Restore a recently deleted task from the Deleted list.
- Deleted tasks remain restorable for 30 days before their identifying content is anonymised.

## My tasks look different on another device

Make sure both devices use the same Apple Account for iCloud and that iCloud is available. Allow time for changes to sync, then reopen to-day on both devices.

Settings can also change what is visible. Compare options such as **Show completed tasks** and **Show archived tasks** in the Settings sheet on each device.

## I cannot find a group

A group appears when at least two visible tasks begin with the same group tag. Check that the tag is at the start of each title and uses a supported format, for example:

```text
[Home] First task
[Home] Second task
```

Also check whether one of the tasks was moved, completed and hidden, archived, or deleted.
